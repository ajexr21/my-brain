### 📝 Part 1: [CEO 지시] 제품 가치 제안 및 MVP 정의 문서화

**프로젝트 가칭:** AJSoft Insight Suite (AI 기반 비즈니스 의사결정 솔루션)
**핵심 Value Proposition:** "흩어져 있는 내부 데이터를 AI가 분석하여, 직관적인 보고서와 실행 가능한 핵심 인사이트를 제공함으로써, 고객사의 의사결정 시간을 획기적으로 단축하고 업무 효율을 극대화합니다."

#### 💡 MVP 목표 (Minimum Viable Product)
*   **범위:** 데이터 연결 및 기초 리포팅 기능에 집중하여, **'데이터 수집 $\rightarrow$ 기본 분석 $\rightarrow$ 시각화 보고서 출력'**의 핵심 파이프라인만 구현합니다.
*   **핵심 성공 지표 (KPI):** 사용자당 월간 활성 보고서 생성 횟수 및 데이터 연결 성공률.

#### ✨ MVP 핵심 기능 목록 및 수익 모델 매핑

| 기능 영역 | 상세 기능 (Feature List) | 가치 제안 (Value Proposition) | Monetization Model 연계 |
| :--- | :--- | :--- | :--- |
| **1. 데이터 통합** | 📊 외부/내부 시스템 API 연결 모듈 (CRM, ERP 등). 자동 스케줄링 및 동기화. | 분산된 데이터를 한곳에 모아 단일 진실의 원천(Single Source of Truth)을 구축합니다. | **Tier Paywall:** 연결 가능한 데이터 소스 수 제한 (Basic: 2개 / Pro: 5개 이상) |
| **2. 핵심 분석/보고** | 📈 자동 지표 추출 및 트렌드 시각화 대시보드. 사용자 정의 KPI 설정 기능. | 복잡한 데이터를 누구나 이해할 수 있는 그래프와 핵심 숫자로 요약합니다. (의사결정 지원) | **Usage-Based:** 월간 분석 보고서 생성량(차트/지표 개수)에 따른 과금. |
| **3. 인사이트 엔진** | 🧠 AI 기반 이상 감지 및 예측 알림 기능 (예: "이번 달 매출 하락 예상"). 핵심 문장 요약 리포팅. | 단순히 보여주는 것을 넘어, '무엇이 잘못되었고 무엇을 해야 할지'를 알려줍니다. (최대 가치) | **Premium Feature:** AI 엔진 사용 횟수 또는 분석 심층도에 따른 높은 구독료 책정. |

#### 💲 가격/번들 옵션 비교 메모 (현빈 개인 목표 반영)

| 옵션명 | 타겟 고객 | 핵심 기능 포함 범위 | 예상 월별 가격대 | 장점 |
| :--- | :--- | :--- | :--- | :--- |
| **1. Basic (Starter)** | 소규모 팀 / 1~2개 데이터 소스만 관리하는 회사 | 기본 대시보드, 수동 보고서 생성, 최대 2개의 데이터 연결. | $49 ~ $79 (낮음) | 진입 장벽이 낮아 초기 고객 확보 용이. **(Lead Magnet 역할)** |
| **2. Pro (Growth)** | 성장 단계의 중소기업 / 핵심 비즈니스 의사결정이 필요한 회사 | Basic 기능 + 자동 스케줄링, 제한적 AI 분석 (이상 감지), 최대 5개 데이터 연결. | $299 ~ $499 (중간) | 가장 많은 기능을 제공하며 수익성이 높음. **(주력 모델)** |
| **3. Enterprise** | 대기업 / 복잡하고 다양한 시스템을 보유한 기업 | Pro 기능 + 무제한 데이터 연결, 커스텀 API 연동, 전담 컨설팅 및 SLA 보장. | $1,000+ (높음) | 가장 높은 ARPU(Average Revenue Per User) 달성 가능. **(최종 목표)** |
